A productivity tool implementing the Pomodoro Technique with customizable work/break intervals and session tracking.

## Features

- **Customizable Intervals**: Set your own work and break durations
- **Multiple Sessions**: Run multiple focus cycles in one go
- **Progress Tracking**: Visual progress bar and time display
- **Audio Notifications**: Bell notifications when phases complete
- **Session Summary**: View total focus time and sessions completed
- **Skip Option**: Press Ctrl+C to skip any phase

## Usage

```bash
python main.py
```

### Configuration

When you start the timer, you'll be prompted to configure:
- **Work Duration**: How long you want to focus (default: 25 minutes)
- **Break Duration**: How long to rest (default: 5 minutes)
- **Number of Sessions**: How many work-break cycles to run (default: 4)

### Example Session

A typical session with defaults (25/5/4):
1. 25 min work + 5 min break
2. 25 min work + 5 min break
3. 25 min work + 5 min break
4. 25 min work (no break after last session)

## Tips for Maximum Productivity

- Set realistic work durations (25-50 minutes)
- Take your breaks seriously - step away from your desk
- Use the notifications to stay aware of phase transitions
- Adjust intervals based on your focus capacity

## Requirements

- Python 3.12+
- No external dependencies
